Buenas! y bienvenido/a a mi repositorio.
Este repositorio contiene un formulario en HTML y CSS con validaciones en JavaScript.  
Incluye campos de nombre, apellido, email, teléfono, fecha de nacimiento, país de residencia y género.
## Cambios realizados.
- Agregue un campo de telefono
- Agregue un campo para genero con sus opciones
- y corregi algunos inconvenientes menores

## Como usar
1. Clonar el repositorio.
2. Abrir "Formulario.html" en un navegador de preferencia.
3. Completar los campos y probar los estilos disponibles.

## Rama utilizada
El trabajo se encuentra en la rama **main** del repositorio.
